# Weather
Weather(날씨) 일기 작성 프로젝트 (학습용 프로젝트)

# 기술 & 툴
- Java jdk 1.8
- MySql
- Spring Boot 2.7.9, gradle
- JAP, Web, Lombok, json-simple 1.1.1, swagger 3.0.0
- JUnit 5
- OpenWeatherMap API 연동
- IntelliJ Ultimate

# 개발 기간
- 2023.03.13 ~ 2023.03.19

# 주요 기능
날씨를 저장하면서 일기를 작성
- 매일 새벽 1시 API 데이터 저장
- 해당 날짜에 맞춰 일기 작성
- 조회 (지정 날짜 기준)
- 조회 (기간 날짜 기준)
- 수정 (지정 날짜 기준)
- 삭제 (지정 날짜 기준)
